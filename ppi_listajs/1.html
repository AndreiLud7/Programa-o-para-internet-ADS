O que é JavaScript?

JavaScript foi criado para tornar as páginas da web mais interativas e dinâmicas. Ele foi desenvolvido inicialmente por Brendan Eich em 1995. Enquanto a execução no navegador permite interações em tempo real com o usuário (como cliques, animações, validações), a execução no servidor (com Node.js, por exemplo) permite que JavaScript também seja usado para processar dados, acessar bancos de dados e servir páginas.

No navegador, o JavaScript roda dentro de uma "sandbox", o que significa que ele tem acesso limitado ao sistema do usuário por questões de segurança.

Alguns objetos globais importantes são:

  • document: permite manipular a estrutura HTML da página.
  • console: usado para registrar mensagens no console do desenvolvedor.
  • alert: exibe caixas de diálogo para o usuário.
